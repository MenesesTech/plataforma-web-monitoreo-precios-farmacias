# 📐 Arquitectura del Sistema de Monitoreo de Precios de Farmacias

## 🧩 Visión General

Este sistema tiene como objetivo monitorear, recolectar y mostrar precios de productos farmacéuticos desde diferentes fuentes. Está compuesto por tres capas principales:

- **Backend:** API REST en Spring Boot
- **Frontend:** Interfaz web en React.js
- **Scraping:** Módulo de extracción de datos con Python + Selenium
